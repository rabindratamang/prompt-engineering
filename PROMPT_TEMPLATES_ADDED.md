## üìã 5 Complete Frameworks Included

### 1. BAB (Before-After-Bridge)
**Best for**: Marketing, persuasive content, transformation stories

**Structure**:
- Before: Current problem situation
- After: Desired outcome
- Bridge: How to get there

**Example Included**: Marketing email for bookkeeping software
- Full prompt with BAB structure
- Complete LLM output (compelling email)
- Analysis of what makes it work

**Key Output**: 
- Clear transformation story
- Emotional connection
- Quantified benefits ($5,000+ savings)
- Strong call to action

---

### 2. RACE (Role-Action-Context-Expectation)
**Best for**: Task delegation, instructions, expert perspective

**Structure**:
- Role: Who the AI should be
- Action: What to do
- Context: Background needed
- Expectation: What good output looks like

**Example Included**: Security code review for healthcare app
- Senior Python developer role
- HIPAA compliance context
- Critical vulnerability identification

**Key Output**:
- Severity ratings (Critical/High/Medium/Low)
- Specific code fixes
- SQL injection and plaintext password issues found
- HIPAA compliance violations listed

---

### 3. CO-STAR (Context-Objective-Style-Tone-Audience-Response)
**Best for**: Complex content, nuanced communication, precise control

**Structure**:
- Context: Background situation
- Objective: What to achieve
- Style: Writing style to emulate
- Tone: Emotional flavor
- Audience: Target reader
- Response: Output format

**Example Included**: Product launch keynote (Steve Jobs style)
- Privacy-first SaaS launching collaboration feature
- Competing with Google Docs
- Steve Jobs keynote style with dramatic pauses

**Key Output**:
- 6:45 minute keynote script
- Dramatic pacing with [PAUSE] markers
- Rebellious tone challenging big tech
- Memorable closing: "Privacy IS progress"

---

### 4. P.A.S (Problem-Agitation-Solution)
**Best for**: Sales copy, cold outreach, urgency-driven content

**Structure**:
- Problem: Identify the issue
- Agitation: Make it feel urgent/painful
- Solution: Present your fix

**Example Included**: Cold email to CTOs about app overload
- 15-30 SaaS tools causing 2+ hours daily waste
- Context switching costs, burnout, shadow IT
- Integration platform solution

**Key Output**:
- Quantified pain ($2M+ productivity drain)
- Multiple pain points (IT, security, CFO, employees)
- Social proof (500+ enterprises)
- Strong P.S. with urgency ($25,000/week cost)

---

### 5. R.T.F (Role-Task-Format)
**Best for**: Quick results, specific output format, simple clarity

**Structure**:
- Role: Who the AI is
- Task: What to do
- Format: Exact output structure

**Example Included**: E-commerce data analysis
- Data analyst role
- Identify top 3 revenue insights
- Specific 4-line format per insight

**Key Output**:
- Exactly 3 insights as requested
- Each with data, action, and quantified impact
- Product B is profit champion ($80/unit)
- Product D has 25% return rate (fix or kill)
- Repeat customers spend 86% more (untapped potential)
